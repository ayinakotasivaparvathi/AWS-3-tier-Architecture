# AWS-3-tier-Architecture Project 🚀
This project demonstrates a 3-tier architecture on AWS using EC2, VPC, RDS, and Load Balancer.
👩‍💻 **Created by:** Siva Parvathi Ayinakota
---
## 📌 Overview

This project demonstrates the implementation of a 3-tier architecture on AWS.
The architecture is divided into three layers: Presentation, Application, and Database to ensure scalability, security, and high availability.
---

![alt text](https://github.com/ayinakotasivaparvathi/AWS-3-tier-Architecture/blob/main/AWS%203tier%20Architecture/Architecture-3%20tier.png)

## ⚙️ AWS Services Used

* Amazon EC2
* Amazon VPC
* Public & Private Subnets
* Internet Gateway
* Route Tables
* Application Load Balancer
* Amazon RDS

---

## 🧩 Implementation Steps

1. Created a VPC to isolate the network.
2. Configured public and private subnets.
3. Attached an Internet Gateway for internet access.
4. Set up Route Tables for traffic routing.
5. Launched EC2 instances for frontend and backend.
6. Deployed the application and tested using public IP.
7. Configured RDS for database management.
8. Implemented Load Balancer for traffic distribution.
